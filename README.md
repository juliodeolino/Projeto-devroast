# devroast

Paste your code. Get roasted.

**devroast** is a code review tool that gives your code brutally honest feedback — with a twist. Toggle "roast mode" for maximum sarcasm, or keep it civil with a standard review. Either way, your code gets scored.

## How It Works

1. **Paste your code** into the editor
2. **Toggle roast mode** for extra sarcasm (or keep it off for a gentler review)
3. **Hit `$ roast_my_code`** and get a score with detailed feedback
4. **Climb (or fall down) the shame leaderboard** — the worst code on the internet, ranked

## Features

- **Code editor** with syntax-aware input and line numbers
- **Roast mode toggle** — brutally honest feedback with maximum sarcasm
- **Score system** — every submission gets rated on a 0-10 scale
- **Shame leaderboard** — the worst-scored code gets immortalized
- **Detailed analysis** — line-by-line issues, suggested fixes with diffs, and category breakdowns

## Getting Started

```bash
# Install dependencies
pnpm install

# Run the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see it in action.
