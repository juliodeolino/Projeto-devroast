# devroast

Cole seu código. Seja roasted.

**devroast** é uma ferramenta de code review que dá feedbacks brutalmente honestos sobre o seu código — com um toque especial. Ative o "roast mode" para o máximo de sarcasmo, ou mantenha desativado para uma análise mais civilizada. De qualquer forma, seu código será avaliado.

## Como Funciona

1. **Cole seu código** no editor
2. **Ative o roast mode** para sarcasmo extra (ou deixe desativado para uma avaliação mais leve)
3. **Clique em `$ roast_my_code`** e receba uma nota com feedback detalhado
4. **Suba (ou desça) no shame leaderboard** — o pior código da internet, ranqueado

## Funcionalidades

- **Editor de código** com input syntax-aware e numeração de linhas
- **Roast mode** — feedback brutalmente honesto com sarcasmo máximo
- **Sistema de pontuação** — cada submissão recebe uma nota de 0 a 10
- **Shame leaderboard** — os piores códigos são imortalizados
- **Análise detalhada** — problemas linha a linha, sugestões de correção com diffs e breakdowns por categoria

## Primeiros Passos

```bash
# Instalar dependências
pnpm install

# Rodar o servidor de desenvolvimento
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) para ver em ação.
